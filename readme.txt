== Description ==
افزونه ورودیار یک پلاگین حرفه‌ای برای مدیریت ورود و ثبت‌نام کاربران وردپرس است. امکانات:
* ورود با ایمیل و نام کاربری
* ثبت‌نام با شماره موبایل و ایمیل
* بازیابی رمز عبور با ایمیل و شماره موبایل
* پنل تنظیمات حرفه‌ای و رنگ‌بندی سفارشی
* شورت‌کدهای اختصاصی
* واکنش‌گرا و مدرن

== Installation ==
1. آپلود فایل ZIP پلاگین به مسیر Plugins → Add New → Upload
2. فعال‌سازی پلاگین
3. تنظیم رنگ، SMS و SMTP در پنل تنظیمات
4. قرار دادن شورت‌کدها در برگه‌ها:
   * `[wy-short-code-login-register]` – فرم ورود و ثبت‌نام
   * `[wy-short-code-password-recovery]` – فرم بازیابی رمز عبور
